# Authentication app with golang(fiber web freamwork) and jwt token
Sample authentication app built with the Fiber framework.

## Stack

- Golang
- Fiber
- Gorm
- Jwt

Quick Start
-----------

```shell
$ git clone https://github.com/reza-nouri/fiber-react-authentication
$ cd fiber
$ go mod tidy
$ go run .
```

## Learn More
